== Description ==

This plugin show an beautiful alert bar in top of your website

== Installation ==

1. Upload `wp-top-alert` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

After completing the above-mentioned steps, wp-top-alert plugin menu appears on the admin menu.
